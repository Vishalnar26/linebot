ROS2 Jazzy on Ubuntu | Arduino IDE | USB serial bridge

## ARCHITECTURE
Arduino → "D:L,C,R|A:L,C,R\n" @ 20Hz → serial_bridge_node → /sensor_data
line_follower_controller_node ← /sensor_data → PID → /cmd_vel → serial_bridge_node → "P:left_pwm,right_pwm\n" → Arduino → L298N

## PIN MAPPING (Arduino Uno)

IN1=6, IN2=7, IN3=8, IN4=9, ENA=5, ENB=10

## PID TUNING (config/pid_params.yaml)
Start: kp=1.0, ki=0.0, kd=0.5, base_speed=0.1
1. Increase kp until robot oscillates, then back off ~20%
2. Add kd (0.3-1.0) to dampen oscillation
3. Add ki only if persistent offset on curves; keep small
4. Raise base_speed once PID is stable
